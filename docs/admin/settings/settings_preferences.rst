.. _settings preferences:

================
``preferences:``
================

.. code:: yaml

   preferences:
     lock:
       - method
       - ..

``lock`` :
  Lock arbitrary settings on the preferences page.

  - autocomplete
  - categories
  - center_alignment
  - doi_resolver
  - favicon_resolver
  - hotkeys
  - image_proxy
  - infinite_scroll
  - language
  - locale
  - method
  - query_in_title
  - results_on_new_tab
  - safesearch
  - search_on_category_select
  - theme
  - url_formatting
